Section 6.3 Explicit deformation spaces (Alex)

¶

Subsection 6.3.1 Set-up

References, Gouvea chapter 5. [1, 25, 26].

We have so far discussed the existence of the universal deformation and some of its properties, now we want to really get our hands on this functor. We do this by relating the residual representation to field extensions.

As usual, \(S\ni p, \infty\) a finite set of places and \(\QQ_S/\QQ\) the maximal extension unramified outside of \(S\text{.}\) Let

\begin{equation*} \Pi = G_{\QQ,S} = \Gal{\QQ_S}{\QQ}\text{.} \end{equation*}

We assume that \(C(\bar\rho )= k\) (or that \(\bar\rho\) is absolutely irreducible).

We can let \(\Pi_0 = \ker(\bar \rho) \subseteq \Pi\) and \(K = \ker(\Pi_0)\) so that we have

If

\begin{equation*} \rho \colon \Pi \to \GL_n(\mathcal R) \end{equation*}

is the universal deformation of \(\bar\rho\) then we can consider

\begin{equation*} \Gamma_n(\mathcal R) = \ker (\GL_n(\mathcal R) \to \GL_n(k)) \end{equation*}

which has the property that

\begin{equation*} \rho(\gamma) \in \Gamma_n(\mathcal R),\,\forall \gamma \in \Pi_0\text{.} \end{equation*}

So we can restrict

\begin{equation*} \rho|_{\Pi_0} \colon \Pi_0 \to \Gamma_n(\mathcal R)\text{.} \end{equation*}
Proposition 6.3.2.

Group theoretically \(\Gamma_n(\mathcal R)\) is a pro-\(p\)-group.

Proof.

This is as

\begin{equation*} \mathcal R = \varprojlim_k \mathcal R/ \ideal m ^k \end{equation*}

so

\begin{equation*} \Gamma_n(\mathcal R) = \varprojlim_k \Gamma_n(\mathcal R/ \ideal m ^k) \end{equation*}

now we can induct using that

\begin{equation*} \Gamma_n(R/\ideal m) = \Gamma_n(k) = \{1\} \end{equation*}

is trivially a \(p\)-group.

Now each transition map

\begin{equation*} \Gamma_n(R/\ideal m ^k) \to \Gamma_ n( R/\ideal m^{k-1}) \end{equation*}

has image a \(p\)-group and the kernel is

\begin{equation*} G_r \coloneqq 1 + M_n(\ideal m^{k-1}/\ideal m^{k}) \end{equation*}

which is isomorphic to

\begin{equation*} M_n(\ideal m^{k-1}/\ideal m^{k})\text{.} \end{equation*}

So in fact the map

\begin{equation*} \rho|_{\Pi_0} \colon \Pi_0 \to \Gamma_n(\mathcal R) \end{equation*}

factors through the maximal (continuous) pro-\(p\) quotient of \(\Pi_0\text{,}\) call this \(P\text{.}\)

This quotient is the galois group of, \(L\text{,}\) the maximal pro-\(p\)-extension of \(K\text{,}\) which is unramified above all primes of \(K\) above \(S\text{,}\) call this set \(S_1\text{.}\)

\begin{equation*} P = \Gal L K\text{.} \end{equation*}

Now letting

\begin{equation*} \tilde \Pi = \Gal L \QQ \end{equation*}

we have the exact sequence

\begin{equation*} 0 \to P \to \tilde \Pi \to \im (\bar \rho) \to 0\text{.} \end{equation*}

So to understand deformations of \(\bar \rho\) we have to understand maps

\begin{equation*} P \to \Gamma_n(R) \end{equation*}

and then how to extend to \(\tilde \Pi\) so that they deform \(\bar\rho\text{.}\)

In order to get a handle on this we make the following assumption:

Definition 6.3.3. Tame residual representations.

A residual representation is tame if

\begin{equation*} p \nmid \# \im(\bar\rho)\text{.} \end{equation*}

The significance of this is that

\begin{equation*} \tilde \Pi \end{equation*}

is a profinite group with a normal \(p\)-Sylow subgroup \(P\text{.}\)

Subsection 6.3.2 Some group theory

Now we recall / learn for the first time some group theoretic results that will be our workhorses today.

If \(G\) is a profinite group with a normal Sylow \(p\)-subgroup \(P\) of finite index that is topologically finitely generated.

Proposition 6.3.4. Schur-Zassenhaus.

With the above notation \(G\) contains a subgroup \(A\) mapping isomorphically onto \(G/P\) (any two such subgroups are conjugate by an element of \(P\)).

This says that for Sylow \(p\)-subgroups all extensions are split (i.e. semidirect products).

Moreover if we let \(\overline P\) be the #

Definition 6.3.5. Frattini quotient.

Let \(P\) be a pro-\(p\)-group, then

\begin{equation*} \overline P \end{equation*}

is the maximal elementary abelian \(p\)-group quotient (recall this is the additive group of a (possibly infinite) vector space over \(\FF_p\)).

This can equivalently be defined as the quotient by the Frattini subgroup, \(\Phi(P)\) of \(P\text{,}\) which is the intersection of all maximal subgroups, and can be obtained as the set of all non-generators (the elements that can be removed from any generating set).

Example 6.3.6.

If \(P\) is the cyclic group \(\ZZ/p^2\) we have the maximal sub \(p\ZZ/p^2\) and the quotient \(\ZZ/p\text{.}\) Or the Heisenberg group

\begin{equation*} \begin{pmatrix} 1 \amp a \amp b \\ 0 \amp 1 \amp c \\ 0 \amp 0 \amp 1\end{pmatrix} \end{equation*}

we get Frattini subgroup

\begin{equation*} \begin{pmatrix} 1 \amp 0 \amp b \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1\end{pmatrix} \end{equation*}

and quotient

\begin{equation*} \begin{pmatrix} 1 \amp a \amp 0 \\ 0 \amp 1 \amp c \\ 0 \amp 0 \amp 1\end{pmatrix}\text{.} \end{equation*}
Proposition 6.3.7. Burnside's basis theorem.

If \(x_1, \ldots, x_d \in P\) generate the quotient \(\overline P\) then they generate \(P\text{.}\)

So we can understand generators of \(P\) by understanding those of \(\overline P\text{!}\)

In fact Boston has strengthened this to

Theorem 6.3.8.

Let \(G\) be a profinite group with a normal \(p\)-Sylow subgroup \(P\) of finite index in \(G\) and \(A\) a subgroup of \(G\) that maps isomorphically to \(G/P\) (note \(A\) is finite). Then if \(A\) acts on \(P\) and \(\overline P\) by conjugation (so that \(\overline P\) is a \(\FF_p\lb A \rb\)-module) and

\begin{equation*} \overline V \subset \overline P \end{equation*}

is an \(\FF_p\lb A \rb\)-submodule. Then there is an \(A\)-invariant subgroup \(V\subseteq P\) which maps onto \(\overline V\) and has \(\dim _{\FF_p}\overline V\) generators.

Example 6.3.9.

If \(\overline V\) is just \(\FF_p\) (written multiplicatively) then an invariant submodule is an

\begin{equation*} \overline x \in \overline P \end{equation*}

for which

\begin{equation*} a\inv \overline x a = \overline x ^{\phi(a)} \end{equation*}

for some character

\begin{equation*} \phi \colon A \to \FF_p^\times \end{equation*}

this then lifts to

\begin{equation*} x\in P : a\inv x a = x ^{\phi'(a)} \end{equation*}

where \(\phi' = \text{teich}\circ \phi\colon A \to \ZZ_p^\times\) .

and also shows

Lemma 6.3.10.

Up to isomorphism there is precisely one semidirect product of \(P\) and \(A\) with given \(A\) action on \(\overline P\text{.}\)

Let \(H \subseteq \GL_n(k)\text{.}\)

Definition 6.3.11.

A \(k\lb H\rb\)-module \(V\) is prime to adjoint if \(\Ad(H) = M_n(k)\) and \(V\) have no irreducible subrepresentations in common.

Let

\begin{equation*} Z_S = \left\{ x \in K^\times : (x) = I^p \text{ for some }I,x_v = y_v^p \in K_v\forall v\in S_1\right\} \end{equation*}

a group for which

\begin{equation*} Z_S\supseteq {K^\times}^p \end{equation*}

so

\begin{equation*} B_S = Z_S/ {K^\times}^p \end{equation*}

which is an elementary abelian \(p\)-group.

As Galois \(H\) acts on \(Z_S, {K^\times}^p\) we have that \(B_S\) is a \(\FF_p\lb H \rb\)-module.

For a pro-\(p\)-group \(P\) Let \(d(P),r(P)\) be ranks of minimal sets of generators and relations for \(P\text{,}\) i.e. the cardinality of a minimal set of generators of \(P\) and relations when viewing \(P\) as a quotient of a free pro-\(p\)-group.

Write \(\delta(F) = 1\) if \(\mu_p \subseteq F\text{,}\) \(\delta(F) = 0\) otw. The significance of these is related to the fact that for certain local fields this is exactly the relation rank!

Theorem 6.3.12. Koch, H., Galoissche Theorie der p-Erweiterungen..
\begin{equation*} r(P) = \left(\sum_{v\in S_1} \delta(K_v) \right) - \delta (K) + \dim_{\FF_p} B_S \end{equation*}
\begin{equation*} d(P) = r_2(K) + 1 + r(P) \end{equation*}

so \(P\) is topologically f.g.

If \(p \nmid h_K\) global class field theory gives a map from the idele class group of \(L\) to \(P\) which induces a map on Frattini quotients and leads to: the exact sequence of \(\FF_p\lb H \rb\)-modules

\begin{equation*} 0 \to B_S \to \overline{\ints_K^\times} \to \bigoplus_{v\in S_1}\overline{\ints_{K_v}^\times} \to \overline P \to 0\text{.} \end{equation*}
Theorem 6.3.13. Boston-Mazur.

If \(p\nmid \# H\) then as \(\FF_p\lb H \rb\)-modules

\begin{equation*} \bigoplus_{v\in S_1} \overline \ints^\times_v \cong \FF_p\lb H \rb \oplus \left( \bigoplus_{l\in S} \Ind_{H_l}^H \mu_p \right) \end{equation*}

sum over decomposition subgroups

\begin{equation*} \overline \ints_K^\times \oplus \FF_p \cong \mu_p \oplus \Ind_{H_\infty}^H \FF_p \end{equation*}

\(H_\infty = \langle \overline \cdot \rangle\text{.}\)

Assume \(\bar \rho\) tame now. \(P\) is normal pro-\(p\)-Sylow subgroup of \(\tilde \Pi\text{,}\) and some \(A \twoheadrightarrow G/P = H\) so \(\tilde \Pi = H \rtimes P\) . As \(\Gamma_n(W(k))\) is pro-\(p\) by Schur-Zassenhaus we have

\begin{equation*} H_1 \subseteq \GL_n(W(k)) \end{equation*}

with \(H \cong H_1\) so we can find a lift

\begin{equation*} \rho_1 \colon \tilde H \to \GL_n(W(k)) \end{equation*}

which induces an isomorphism

\begin{equation*} A \xrightarrow\sim H_1 \end{equation*}

we denote

\begin{equation*} \sigma \colon A \to \GL_n(W(k))\text{.} \end{equation*}

Given a coefficient ring \(R\) we have

\begin{equation*} W(k) \to R \end{equation*}

so

\begin{equation*} \sigma_R \colon A \to \GL_n(R) \supseteq \Gamma_n(R) \end{equation*}

and \(A\) acts on \(\Gamma_n(R)\) via conjugation.

Let

\begin{equation*} E_{\bar \rho} \colon \mathcal C \to \Set \end{equation*}
\begin{equation*} E_{\bar \rho} (R) = \Hom_A(P, \Gamma_n(R))\text{.} \end{equation*}

Then we have a natural morphism

\begin{equation*} E_{\bar \rho} \to D_{\bar \rho} \end{equation*}

given by

\begin{equation*} \phi\in E_{\bar \rho}(R) \mapsto\tilde \Pi = A \rtimes P \xrightarrow{\sigma_R, \phi} \GL_n(R)\text{.} \end{equation*}
Theorem 6.3.14. Boston.

\(R_{\bar \rho}\) is representable.

  1. If \(C(\bar \rho) = k\) then
    \begin{equation*} E_{\bar \rho} \xrightarrow \sim D_{\bar \rho}\text{.} \end{equation*}
  2. Otherwise this morphism is smooth and induces an isomorphism of tangent spaces.
Theorem 6.3.15. Boston.

Let \(p\) be odd, and

\begin{equation*} \bar \rho \colon G_{\QQ, S} \to \GL_2(\FF_p) \end{equation*}

odd, absolutely irreducible and tame, \(H = \im(\bar\rho)\text{.}\) Let

\begin{equation*} K = \QQ_{S}^{\ker \bar \rho} \end{equation*}
\begin{equation*} V = \coker\left( \mu_p(K) \to \prod_{v \in S_1} \mu_p(K_v)\right) \end{equation*}

both \(V, B_S\) are \(\FF_p\lb H \rb\) modules, suppose \(p\nmid h_K\) and \(V,B_S\) are relatively prime to \(\Ad(\bar \rho)\) as \(\FF_p\lb H \rb\)-modules, then

\begin{equation*} R(\bar\rho) \cong \ZZ_p \lb\lb T_1, T_2, T_3\rb\rb\text{,} \end{equation*}

moreover \(\rho\) can be described explicitly.

Proof.

Using Theorem 6.3.13 we have that \(\overline P\) is generated by

  1. \(\overline x\) fixed by \(H\)
  2. \(\overline y\) with \(\overline y ^c = \overline y \inv\)
  3. other elements generating a prime-to-adjoint \(\FF_p\lb H \rb\)-module.

by Theorem 6.3.8 we get that \(P\) is generated by some \(x\) fixed by \(H\) and \(y\) s.t. \(y^c = y\inv\) and other prime-to-adjoint gens. So the dimension of the tangent space is 3 and we the universal deformation to

\begin{equation*} \ZZ_p\lb \lb T_1, T_2, T_3\rb \rb \end{equation*}

via

\begin{equation*} x\mapsto \begin{pmatrix} 1+ T_1\amp 0 \\ 0 \amp 1 + T_1\end{pmatrix} \end{equation*}
\begin{equation*} y\mapsto \begin{pmatrix} (1+ T_2T_3)^{\frac12}\amp T_2 \\ T_3 \amp (1+ T_2T_3)^{\frac12}\end{pmatrix} \end{equation*}

other generators mapping to 0.

Where do these matrices come from? We are trying to write down the universal matrices which are fixed by conjugation by any element, or are inverted after conjugation by complex conjugation.

For the first the only matrices in the center are multiples of the identity which gives the form of \(x\text{.}\) For the second if we take

\begin{equation*} c = \begin{pmatrix} 1 \amp 0 \\ 0 \amp -1\end{pmatrix} \end{equation*}

then \(c y c\inv = y\inv\) has the following consequence. The condition gives \(\det y = \pm 1\text{.}\) As \(cyc\inv\) is \(y\) with the off diagonal elements negated assuming \(\det = 1\) the inverse is the same with the diagonal terms swapped: we see that the two terms on the diagonal must be equal and we can choose the off diagonal terms to be whatever we like, call them \(T_2, T_3\text{.}\) Writing out what \(\det = 1\) means for equal diagonal terms shows both are \((1+ T_2 T_3)^{\frac 12}\text{.}\)
