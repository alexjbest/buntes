For \(L\) a line bundle on \(A\) we get a map

\begin{align*} \phi_L\colon A(K) \amp\to \Pic(A)\\ a\amp\mapsto t_a^* L\otimes L^{-1} \end{align*}

where

\begin{equation*} \Pic(A) = \{\text{line bundles on } A\}/\sim\text{.} \end{equation*}
