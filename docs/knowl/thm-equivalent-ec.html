Theorem 1.2.5

TFAE for a projective curve \(E\) over \(k\text{.}\)

  1. \(E\) is given by \(Y^2Z = X^3 + aXZ^2 + bZ^3\text{,}\) \(4a^3 + 27b^2 \ne 0\text{.}\)
  2. \(E\) is nonsingular of genus 1 with a distinguished point \(P_0\text{.}\)
  3. \(E\) is nonsingular with an algebraic group structure.
  4. (if \(k \subseteq \CC\)) such that \(E(\CC) = \CC/\Lambda\) for some lattice \(\Lambda \subseteq \CC\text{.}\)
