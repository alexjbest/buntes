Let's try and do an \(X_0(37)\) example. This time we are lucky as \(37\) splits in \(\QQ(\sqrt{-3})\) so we should be able to find an order of \(M_0(37)\) isomorphic to \(\ZZ\lb (1 + \sqrt{-3})/2\rb\text{.}\) If I wasn't so lazy I would think some more about how to do this in general, but matching the charpoly of a matrix of the form

\begin{equation*} \begin{pmatrix} \ast \amp \ast \\ 37 \amp \ast \end{pmatrix} \end{equation*}

to \(x^2 - x + 1\) (the charpoly of \((1+ \sqrt{-3})/2\) we find)

\begin{equation*} \begin{pmatrix} -9 \amp -3 \\ 37 \amp 10 \end{pmatrix} \end{equation*}

so

\begin{equation*} \ints \cong \ZZ + \ZZ \begin{pmatrix} -9 \amp -3 \\ 37 \amp 10 \end{pmatrix}\text{,} \end{equation*}

a \(\tau\) fixed by this is

\begin{equation*} \tau = \frac{\sqrt{-3} - 21}{74} \end{equation*}

so letting \(q= e^{2 \pi i \tau}\) we compute

\begin{equation*} \sum_{n =1}^{1000} \frac{a_n}{n} q^n \end{equation*}

using the anlist of \(37a1\) to get

\begin{equation*} 0.204680500375773 - 1.22569469099339i \end{equation*}

so we use Sage's elliptic_exponential to find

\begin{equation*} (-1.00000000000000 + 1.77644036369769e-15I : 2.22044604925031e-16 + 3.55288072739538e-15I : 1.00000000000000) \end{equation*}

so we have discovered the point \((-1, 0 , 1)\) on

\begin{equation*} y^2 + y = x^3 - x\text{,} \end{equation*}

amazing.
