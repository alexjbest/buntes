Item 2
The modular equation
\begin{equation*} \phi_l (x,y) = 0 \end{equation*}
is a canonical model for
\begin{equation*} Y_0(l) = \Gamma_0(l) \backslash \HH \end{equation*}
it parameterises pairs of elliptic curves related by an \(l\)-isogeny. This moduli interpretation is still valid when we use any field \(F\) with \(\characteristic(F) \ne l\text{.}\)
