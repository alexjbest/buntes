\(L\in \Pic^0(A)\) and \(m,p_1,p_2\colon A\times A \to A\)

  1. \begin{equation*} m^* L \cong p_1^* L\otimes p_2^* L \end{equation*}
  2. Given \(f,g \colon X \to A\)
    \begin{equation*} (f+g)^* L \cong f^* L \otimes g^* L \end{equation*}
  3. \begin{equation*} [n]^* L \cong L^{\otimes n} \end{equation*}
  4. \begin{equation*} \phi_L(A(k)) \subseteq \Pic^0(A) \end{equation*}
    for \(L \in \Pic (A)\text{.}\)
