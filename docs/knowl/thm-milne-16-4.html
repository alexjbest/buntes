Theorem 1.12.4 16.4

Let \(g= \dim (X)\text{.}\)

  1. Every root of \(f_X\) \(\alpha \in \CC\) satisfies \(|\alpha| = q^{1/2}\text{.}\)
  2. If \(\alpha\) is a root of \(f_X\text{,}\) then \(\bar \alpha\) with the same multiplicity. In particular if \(\alpha = \pm \sqrt q\) then it occurs with even multiplicity.
