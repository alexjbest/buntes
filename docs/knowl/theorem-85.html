Theorem 5.8.10 Weil 1956, Milne 14.6

\(X/\CC\) descends to \(k\) if all \(X^\sigma \simeq X\) and there exists a set of points \(P_1,\ldots, P_n\in X(\CC)\) s.t.

  1. The only automorphism of \(X\) fixing each \(P_i\) is the identity.
  2. There exists a subfield \(L \subseteq \CC\) finitely generated \(/k\) s.t. \(\sigma P_i = P_i\) for all \(\sigma\) fixing \(L\text{.}\)
