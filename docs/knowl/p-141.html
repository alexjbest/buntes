Let \(X\) be an abelian variety and \(L\) a line bundle, \(L = L(D)\) then TFAE:

  1. \(H(D) = \{x\in X: T_x^*D = D \}\) is finite.
  2. \(K(L) = \{x\in X: T_x^*L \cong L \}\) is finite.
  3. \(|2D|\) is basepoint free and defines a finite morphism \(X\to \PP^N\text{.}\)
  4. \(L\) is ample.
