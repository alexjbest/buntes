If \(\overline V\) is just \(\FF_p\) (written multiplicatively) then an invariant submodule is an

\begin{equation*} \overline x \in \overline P \end{equation*}

for which

\begin{equation*} a\inv \overline x a = \overline x ^{\phi(a)} \end{equation*}

for some character

\begin{equation*} \phi \colon A \to \FF_p^\times \end{equation*}

this then lifts to

\begin{equation*} x\in P : a\inv x a = x ^{\phi'(a)} \end{equation*}

where \(\phi' = \text{teich}\circ \phi\colon A \to \ZZ_p^\times\) .
