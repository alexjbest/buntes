If

\begin{equation*} \rho \colon \Pi \to \GL_n(\mathcal R) \end{equation*}

is the universal deformation of \(\bar\rho\) then we can consider

\begin{equation*} \Gamma_n(\mathcal R) = \ker (\GL_n(\mathcal R) \to \GL_n(k)) \end{equation*}

which has the property that

\begin{equation*} \rho(\gamma) \in \Gamma_n(\mathcal R),\,\forall \gamma \in \Pi_0\text{.} \end{equation*}

So we can restrict

\begin{equation*} \rho|_{\Pi_0} \colon \Pi_0 \to \Gamma_n(\mathcal R)\text{.} \end{equation*}
