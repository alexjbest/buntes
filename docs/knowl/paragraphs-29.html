Vertical isogenies

Let \(\ints\) be an order in an imaginary quadratic field \(K \) of discriminant \(D \lt -4\) and let \(\ints' = \ZZ+l\ints\text{.}\)

Lemma 3.1.14

Let \(E' / k\) be an elliptic curve with CM by \(\ints '\) then there is a unique ascending \(l\)-isogeny \(E'\to E\) with \(E/k\) an elliptic curve with CM by \(\ints\text{.}\)

Definition 3.1.15

An \(l\)-volcano \(V\) is a connected undirected graph whose vertices are partitioned into levels \(V_0, \ldots, V_d\text{.}\)

  1. The subgraph \(V_0\) is regular of degree \(\le 2\text{.}\)
  2. For each \(i \gt 0\) each vertex in \(V_i\) has exactly one neighbour in level \(V_{i-1}\text{,}\) and this accounts for all edges outside of \(V_0\text{.}\)
  3. For \(i \lt d\) each vertex has degree \(l+1\text{.}\)

The number \(d\) is the depth.

Theorem 3.1.17 Kohel

Let \(V\) be an ordinary component of \(G_l(\FF_q)\) that doesn't contain \(0\) or \(1728\) then \(V\) is an \(l\)-volcano s.t.

  1. All vertices in \(V_i\) have the same endomorphism ring \(\ints_i\text{.}\)
  2. The subgraph on \(V_0\) has degree
    \begin{equation*} 1+ \left(\frac {\disc(K)}{l}\right) \end{equation*}
    where \(K = \Frac(\ints_0)\)
  3. If
    \begin{equation*} \left(\frac {\disc(K)}{l}\right) \ge 0 \end{equation*}
    then \(\#V_0\) is the order \(\lb l \rb\) in \(\Cl (\ints_0)\) else \(\#V_0 = 1\text{.}\)
  4. The depth of \(V\) is \(d = v_l (\lb \ints_K : \ZZ\lb\pi \rb\rb)\) where \(\pi\) is the Frobenius morphism on any \(E\) with \(j(E) \in V\text{.}\)
  5. \(l \nmid \lb \ints_K : \ints_0 \rb\text{,}\) \(\lb \ints_i : \ints_{i+1}\rb = l\) for \(0\le i \lt d\text{.}\)
