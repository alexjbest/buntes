Let

\begin{equation*} Z_S = \left\{ x \in K^\times : (x) = I^p \text{ for some }I,x_v = y_v^p \in K_v\forall v\in S_1\right\} \end{equation*}

a group for which

\begin{equation*} Z_S\supseteq {K^\times}^p \end{equation*}

so

\begin{equation*} B_S = Z_S/ {K^\times}^p \end{equation*}

which is an elementary abelian \(p\)-group.
