Fact 3.1.5
  1. If \(p \nmid n = \deg \phi\) then the kernel of \(\phi\) has size \(n\) (\(\phi\) is separable).
  2. every finite subgoup of \(E(\overline k )\) is the kernel of a separable isogeny from \(E\text{,}\) unique up to isomorphism.
  3. Every \(n\)-isogeny \(\phi\colon E \to E'\) has a dual isogeny \(\hat \phi \colon E' \to E\) such that
    \begin{equation*} \phi\circ \hat\phi = \hat\phi \circ \phi = \lb n \rb\text{,} \end{equation*}
    the multiplication-by-\(n \) map.
  4. The \(n\)-torsion subgroup
    \begin{equation*} E\lb n \rb = \left\{ P \in E(\overline k) : nP = \infty\right\} \end{equation*}
    is isomorphic to \((\ZZ/n)^2\) if \(p\nmid n\text{.}\)
