The Gauss norm / sup norm

\begin{equation*} |\sum a_J x^J | = \max_J |a_J| \ge 0 \end{equation*}

properties

  1. \begin{equation*} |f| = 0 \iff f = 0 \end{equation*}
  2. \begin{equation*} |cf| = |c|_k |f| \end{equation*}
  3. \begin{equation*} |f + g| \le \max\{|f|, |g|\} \end{equation*}
  4. \begin{equation*} |fg| = |f||g| \end{equation*}
