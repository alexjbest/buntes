There are limitations to the above coming from the Heegner hypothesis, which meant that we could only use fields \(K\) in which all primes dividing \(N\) split. This limitation can be removed, and instead we may work with any \(K\) whose discriminant is coprime to \(N\) (so we allow inert things now too), at the expense of working with Shimura curves instead of modular curves.
