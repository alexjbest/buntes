The modular equation

Let \(j(\tau)\) be the modular \(j\)-function. For each prime \(l\) the minimal polynomial \(\phi_l\) of \(j(l\tau)\) over \(\CC (j(\tau))\) is the modular polynomial

\begin{equation*} \phi_l \in \ZZ[j(\tau)][y] \simeq \ZZ[x,y]\text{.} \end{equation*}
Fact 3.1.7
  1. \(\phi_l\) is symmetric in \(x,y\) and has a degree \(l+1\) in both variables.
  2. The modular equation
    \begin{equation*} \phi_l (x,y) = 0 \end{equation*}
    is a canonical model for
    \begin{equation*} Y_0(l) = \Gamma_0(l) \backslash \HH \end{equation*}
    it parameterises pairs of elliptic curves related by an \(l\)-isogeny. This moduli interpretation is still valid when we use any field \(F\) with \(\characteristic(F) \ne l\text{.}\)
  3. Let \(m_l(j,j') = \ord_{t = j'} \phi_l(j,t)\text{,}\) whenever \(j,j' \ne 0,1728\text{,}\)
    \begin{equation*} m_l(j,j') = m_l(j',j)\text{.} \end{equation*}
