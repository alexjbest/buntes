Proof

Using Theorem 6.3.13 we have that \(\overline P\) is generated by

  1. \(\overline x\) fixed by \(H\)
  2. \(\overline y\) with \(\overline y ^c = \overline y \inv\)
  3. other elements generating a prime-to-adjoint \(\FF_p\lb H \rb\)-module.

by Theorem 6.3.8 we get that \(P\) is generated by some \(x\) fixed by \(H\) and \(y\) s.t. \(y^c = y\inv\) and other prime-to-adjoint gens. So the dimension of the tangent space is 3 and we the universal deformation to

\begin{equation*} \ZZ_p\lb \lb T_1, T_2, T_3\rb \rb \end{equation*}

via

\begin{equation*} x\mapsto \begin{pmatrix} 1+ T_1\amp 0 \\ 0 \amp 1 + T_1\end{pmatrix} \end{equation*}
\begin{equation*} y\mapsto \begin{pmatrix} (1+ T_2T_3)^{\frac12}\amp T_2 \\ T_3 \amp (1+ T_2T_3)^{\frac12}\end{pmatrix} \end{equation*}

other generators mapping to 0.
